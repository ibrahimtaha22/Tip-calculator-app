Frontend Mentor - Tip calculator app
This is a solution to the Tip calculator app challenge on Frontend Mentor. This project calculates the tip and total bill amount per person, featuring a fully responsive design.

Features
Users are able to:

View the optimal layout for the app depending on their device's screen size (Responsive Design).

See hover and focus states for all interactive elements.

Calculate the correct tip and total cost of the bill per person based on inputs.

Experience form validation (e.g., an error message appears when entering "0" in the number of people field).

Built with
Semantic HTML5 markup: For a structured and accessible document.

SCSS: Using variables, mixins, and nesting for organized and maintainable styling.

JavaScript: To handle the application logic and real-time result calculations.

Flexbox & CSS Grid: Used for a precise and flexible layout.

What I learned
Through this project, I practiced several key frontend skills:

Handling Input Events: Learning how to monitor user input and update the UI dynamically without needing a page refresh.

Form Validation: Implementing logic to check for invalid inputs (like zero) and providing visual feedback to the user.

Pixel-Perfect Design: Developing an eye for detail to match a static image design as closely as possible using CSS.
